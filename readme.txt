== Description ==

Allows a user to create a post on a WordPress Site which allows an end user to create a decision tree / series of choices which an end user can work through.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the contents of the zip (including the folder) to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the settings page for the plugin - No other configuration is required.
